# Mission Leader

A web app for role assignment in [The Resistance](https://boardgamegeek.com/boardgame/41114/resistance). This tool handles secret role distribution with special roles from the expansion. You still play the actual game in person.

**Note:** You need the base game and expansion to play.

## Getting Started

```bash
uv sync
uv run python run.py
```
